## Как использовать

Опишите как пользователю запустить ваш проект шаг за шагом:

1. Клонируйте репозиторий:

```bash

cd ВАШ_ПРОЕКТ
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Запустите `main.py`:

```bash
python main.py
```
